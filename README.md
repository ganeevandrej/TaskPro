# TaskPro

## О проекте

TaskPro — учебный full-stack проект, разработанный в рамках выпускной квалификационной работы. Приложение ориентировано на управление личными задачами, организацию рабочего процесса и использование техник повышения продуктивности.

Проект состоит из мобильного клиента на `React Native` и серверной части на `Node.js`. Клиентское приложение взаимодействует с API, получает данные о задачах, категориях и уведомлениях, а сервер отвечает за бизнес-логику, авторизацию, работу с базой данных и обработку пользовательских запросов.

## Скрипты

### Backend

```bash
npm start
npm run dev
```

### Frontend

```bash
cd client
npm start
npm run android
npm run ios
npm run web
```

## Структура проекта

```text
taskPro/
├─ client/           # Мобильное приложение
├─ controllers/      # Контроллеры серверной логики
├─ service/          # Сервисный слой и бизнес-логика
├─ routes/           # API-маршруты
├─ middlewares/      # Промежуточные обработчики
├─ dtos/             # DTO-модели
├─ exceptions/       # Обработка ошибок
├─ db/               # Подключение к базе данных
├─ uploads/          # Загруженные файлы
├─ server.js         # Точка входа backend
└─ package.json
```

## Чему посвящен проект

Проект посвящен теме личной эффективности и управления задачами. В его основе лежит идея создания мобильного инструмента, который помогает пользователю планировать дела, распределять задачи по категориям, отслеживать выполнение, получать уведомления и использовать популярные техники тайм-менеджмента в одном приложении.
