# План развития проекта TaskPro в Fullstack направлении

## 🎯 Цель проекта

Превратить учебный проект (ВКР) в полноценный fullstack проект, демонстрирующий профессиональные навыки в:
- Backend разработке (Node.js, Express, PostgreSQL)
- Frontend разработке (React Native/Expo)
- Безопасности и лучших практиках
- Тестировании
- DevOps и деплое
- Архитектуре и паттернах проектирования

---

## 📊 Текущее состояние проекта

### Что уже реализовано:
✅ Express.js API с аутентификацией  
✅ React Native мобильное приложение  
✅ PostgreSQL база данных  
✅ JWT токены для авторизации  
✅ Push-уведомления  
✅ CRUD операции для задач, категорий, техник  
✅ Базовое управление задачами  

### Что требует улучшения:
❌ Критические уязвимости безопасности  
❌ Отсутствие тестов  
❌ Нет деплоя и CI/CD  
❌ Слабая архитектура (нет репозиториев, смешанная ответственность)  
❌ Отсутствие документации  
❌ Хардкод конфигураций

## 🔄 Процесс работы

### Еженедельный ритм:
1. **Понедельник:** Планирование задач на неделю из плана
2. **Вторник-Четверг:** Реализация задач
3. **Пятница:** Тестирование, коммиты, подведение итогов
4. **Выходные:** Отдых или легкие задачи (документация)

### Коммиты:
- Используйте conventional commits (`feat:`, `fix:`, `refactor:`, `test:`, `docs:`)
- Коммитьте часто, маленькими порциями
- Пишите понятные commit messages

### Pull Requests:
- Создавайте PR даже для своих изменений
- Используйте PR для code review (можно само-ревью)
- Связывайте PR с задачами из плана

## 💡 Советы по успешному выполнению

1. **Начинайте с критичного:** Этап 1 обязателен, без него проект нельзя считать безопасным
